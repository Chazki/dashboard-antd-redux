# dashboard-antd-redux

## Intructions:

### Client:

- Open a console and run the next commands:
  - `cd Desktop`
  - `git clone https://github.com/BryanVe/dashboard-antd-redux.git`
  - `cd dashboard-antd-redux`
  - `cd client`
- Then, open `.env` file and put your google maps api key.
- Finally run:
  - `npm install`
  - `npm start`
